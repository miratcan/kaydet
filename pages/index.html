Terminal Native

Capture ideas without leaving your shell. Kaydet honours your $EDITOR, plays nicely with git, and stores everything in timestamped text files.

Structured On Save

Tags, metadata, and words are indexed automatically. Combine natural text with filters like time:>2 or status:done to surface anything instantly.

Edit & Delete by ID

Adjust any entry using kaydet --edit 42 or remove it with kaydet --delete 42. IDs stay stable, files and SQLite index stay in sync.

AI-Ready Output

JSON responses, tag counts, and an optional MCP server make Kaydet ready for automation, dashboards, or your favourite AI copilot.

Install & Capture in Seconds

Choose the workflow that matches your toolbox. Kaydet takes less than a minute to install and start logging.

  • pip install kaydet
  • pipx install git+https://github.com/miratcan/kaydet.git
  • pip install "kaydet[mcp]" for Model Context Protocol integrations

Build Narratives, Not Just Notes

Kaydet keeps your diary, work log, and learning journal in one searchable archive. It’s lightweight enough for a thought, powerful enough for a personal knowledge base.
